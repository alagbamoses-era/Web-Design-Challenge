Summary: Latitude vs. X

The purpose of this project was to analyse how weather changes as one approaches the equator. To accomplish this analysis, we first pulled data from the OpenWeatherMap API to assemble a dataset on over 500 cities. After assembling the dataset, we used Matplotlib to plot various aspects of the Weather vs Latitude. Factors we looked at included temperature, cloudiness, humdity and wind speed. This site provides the source data and visualizations created as part of the analysis as well as expalnations and descriptions of any trends and correlations that existed.
